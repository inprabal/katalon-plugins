# KATALON PLUGINS #  

## Gradle Build ##  

Katalon 8x with jdk11+ use **build_katalon_8x.gradle**   

		gradle -Pkatalon8 katalonCopyDependencies   

Katalon 9x with jdk11+ use **build_katalon_9x.gradle** 

		gradle katalonCopyDependencies    
	
To packege plugins:   
        gradle katalonPluginPackage
